# Satsuma Ventures

The website and brand assets for [Satsuma Ventures](https://satsumaventures.com) — a private family venture studio based in Seattle.

> *A family venture studio cultivating ideas worth growing.*

## Repository contents

- [`index.html`](index.html) — the live landing page (served at satsumaventures.com)
- [`brand.html`](brand.html) — the full brand book & design system
- [`assets/`](assets/) — logo files (SVG, light and dark variants)
- [`design-system/`](design-system/) — design tokens, CSS, preview cards, and an Agent Skills entry point for AI-assisted design work against the Satsuma brand
